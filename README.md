# code-refactor-agent-demo

这是一个个人学习项目：基于 OpenClaw + DeepSeek V4 的代码重构建议 Agent。

## 解决的问题
个人写 Python 脚本时，手动找重复代码和过长函数耗时费力。

## 核心逻辑流
1. 解析代码结构
2. 按优先级识别问题（重复块 > 长函数 > 缺异常）
3. 生成重构建议和代码
4. 语法自检

## 运行示例（截图见下方）
（由于未部署 web，这是本地测试输出）

> 输入文件：test.py（包含一个120行的函数）
> Agent输出：
> - 发现函数过长，建议拆分为3个子函数
> - 发现缺少 try/except，建议添加异常捕获
> - 重构后代码行数减少18%
